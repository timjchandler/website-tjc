Sentencing Calculator

This is an expert system build for a practical course during my degree. The model is built in Java, using a knowledge base written in XML. The GUI uses the JavaFX library. The full system and its documentation may be found on my github.

The system itself could only serve an extremely niche purpose, and is only aimed to be a proof of concept for an expert system.
